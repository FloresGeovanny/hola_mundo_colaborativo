Esta es una practica colaborativa que escribe un "hola mundo" el cual consiste en los siguientes códigos:

1.-Creación de README

2.-Algoritmo(Pseint):
Es un lenguaje de programación estructurada que define y demuestra la sintaxis básica de pseint el cual te da un hola mundo con un print"hola mundo"

2.-Programacion estructurada:
Este programa imprime "hola mundo" en la pantalla mostrando  como utilizar la función print() en python que sirve para crear programas que siguen una lógica secuencial y estructurada, ideal para tareas simples y complejas 

3.-Diagrama de clases:
Muestra la estructura básica de la clase HolaMundo, incluyendo sus métodos y atributos, la clase tiene un constructor y un método saludar() que imprime "Hola Mundo" en la pantalla.

4.-Programacion orientada a objetos:
El programa define una clase HolaMundo que tiene un método saludar() que imprime "Hola Mundo" en la pantalla, demostrando los conceptos básicos de la programación orientada a objetos que es de forma mas organizada y mas ajustable 

5.-Programacion con interfaz grafica (tkinter y clases):
El programa crea una ventana gráfica con un label que dice "Hola Mundo", demostrando cómo crear interfaces gráficas de usuario con Tkinter.
